# Visa Travels - Intentional Booking App

A premium, fully functional travel booking landing page built using modern Semantic HTML5, Vanilla CSS, and Vanilla JavaScript.

## Features
- **Visual Imagery**: Generated custom landscapes in `images/` representing top destinations.
- **Glassmorphic Filter Bar**: Floating panels for real-time, responsive query filtering.
- **Values Carousel**: Offset grid cards supporting click navigations and swipe-to-drag gestures.
- **Animate Track Scroll**: Vertical timeline navigation keeping track of active viewports.
- **Interactive Quiz Dialog**: Matches travelers to their ideal trip based on personality answers.
- **Booking Dialog Invoice**: Validates guest credentials and outputs custom receipt details.

## How to Run Locally
Ensure you have a local HTTP server. You can run one using:
```bash
# Using Ruby (included on macOS)
ruby -run -e httpd . -p 8000
```
Then visit [http://localhost:8000](http://localhost:8000) in your web browser.
